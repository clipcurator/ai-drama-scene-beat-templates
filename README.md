# AI Drama Scene Beat Templates

Scene beats are the smallest useful units of drama structure. They help creators decide what changes inside a scene: what the character wants, what blocks them, what new information appears, and what emotional shift happens before the next cut. This repository collects reusable scene beat templates for AI drama, vertical short drama, and script-to-video workflows.

It is designed for creators who need scene-level structure before writing full dialogue or generating storyboards.

## What is a scene beat?

A scene beat is a meaningful shift inside a scene. It can be:

- a power change
- a reveal
- a refusal
- a new obstacle
- an emotional turn

In short drama, scenes need beats quickly because runtime is limited.

## Beat patterns

| Pattern | Best for | Internal movement |
|---|---|---|
| Confrontation beat | Accusation, tension, rivalry | claim -> resistance -> evidence -> shift |
| Reveal beat | Secret, identity, betrayal | hint -> denial -> proof -> new truth |
| Emotional beat | Romance, family, betrayal | defense -> wound -> vulnerability -> choice |
| Pressure beat | Deadline, threat, power | request -> refusal -> pressure -> cost |
| Exit hook beat | End-of-scene suspense | conflict -> partial answer -> stronger question |

## Review workflow

1. Define what each character wants in the scene.
2. Identify the blocking force.
3. Decide what changes by the end of the scene.
4. Make the change visible in action, dialogue, or evidence.
5. End with a useful next beat or hook.
