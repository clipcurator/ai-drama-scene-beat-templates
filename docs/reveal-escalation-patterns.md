# Reveal and Escalation Patterns

## Hint -> denial -> proof -> shift

Use when a character hides a truth until evidence forces change.

## Demand -> refusal -> threat -> cost

Use when power or urgency needs to rise inside one scene.

## Soft moment -> interruption -> new danger

Use when you want to break intimacy with conflict.

## Public accusation -> private truth

Use when the surface conflict hides a deeper emotional issue.

## Partial reveal -> bigger question

Use when the scene should end with a stronger hook than it started with.
