# AI 短剧情节拍模板库

情节拍是场景里最小但有用的戏剧结构单位。它帮助创作者决定：角色在场景里想要什么、什么在阻拦他、出现了什么新信息、以及切到下一幕前发生了什么情绪变化。本仓库整理 AI 短剧、竖屏短剧和 script-to-video 工作流中的可复用场景 beat 模板。

它适合在写完整对白或生成分镜前，先搭建场景级结构的创作者。

## 什么是场景 beat

场景 beat 指场景内部发生的一次有效变化。它可以是：

- 权力变化
- 真相揭示
- 拒绝
- 新阻碍
- 情绪转折

短剧时长有限，所以场景必须更快出现 beat。

## Beat 模式

| 模式 | 适合场景 | 内部运动 |
|---|---|---|
| 对峙 beat | 指控、张力、对手戏 | 主张 -> 抵抗 -> 证据 -> 变化 |
| 揭示 beat | 秘密、身份、背叛 | 暗示 -> 否认 -> 证明 -> 新真相 |
| 情绪 beat | 爱情、家庭、背叛 | 防御 -> 伤口 -> 脆弱 -> 选择 |
| 压力 beat | 截止、威胁、权力 | 请求 -> 拒绝 -> 施压 -> 代价 |
| 出口悬念 beat | 场景结尾 suspense | 冲突 -> 局部回答 -> 更强问题 |

## 评估流程

1. 先定义场景中每个角色想要什么。
2. 确定阻拦力量来自哪里。
3. 明确场景结束时到底改变了什么。
4. 让变化在动作、对白或证据中被看见。
5. 用下一拍或悬念收尾。
